math

                        Example: sbr-math 2+2/3
                        Example: /math type:square root num1:2
                        Example: /math type:factorial num1:6
Aliases: solve

Options:
Name Type Required Description Choices/limits/formatting Default
Equation String True The equation to solve. Message command only N/A N/A
Type String True The type of equation to solve square/square root
factorial
HCF/LCM
AR if DT/HT/ms
circumference/area of a circle
pythagorus
convert to sig fig
OD if DT/HT/ms
ms to AR/OD
Mod int to string

Num1 Float True The first number in the equation N/A N/A
Num2 Float False The second number in the equation N/A N/A

map

Returns the information of a beatmap.
The buttons at the bottom switch between difficulties.
Sending the map link or .osu file will also work.

                        Command: sbr-map "[search]" [id] +[mods]
                  Slash Command: /map [id] [mods] [detailed]
                        Example: sbr-m "harumachi" +HDDT
                        Example: /map mods:DT detailed:true
                        Link (full): https://osu.ppy.sh/beatmapsets/[sid]#[gamemode]/[id] +[mods]
                  Link (short, map): https://osu.ppy.sh/b/[id] +[mods]
                  Link (short, set): https://osu.ppy.sh/s/[sid] +[mods]
                   Link (long, map): https://osu.ppy.sh/beatmap[id] +[mods]
                   Link (long, set): https://osu.ppy.sh/beatmapsets/[sid] +[mods]
                        File: [filename].osu
Aliases: m

Options:
Name Type Required Description Choices/limits/formatting Default
Search String False The search query for the map. ie artist, title, difficulty, etc. Overrides map ID N/A null
ID Integer False The id of the map. If omitted, the previous map will be used instead N/A Previous map
Mods String False Mods to use for difficulty/pp calculations N/A NM
Detailed Boolean False Shows extra map details (ar in ms, od in ms, object radius etc.) True/False False
SID Integer Sometimes The ID of the beatmap set.
Required if using osu.ppy.sh/s/ or osu.ppy.sh/beatmapsets/		 N/A N/A
Mode String Sometimes The mode of the beatmap.
Required if using full map link		 osu/taiko/fruits/mania osu

osutop

Retrieves the top plays of the user.
The buttons at the bottom switch between pages.
Maximum of 5 scores shown per page (9 if compact is enabled).

                        Command: sbr-osutop [user]
                  Slash Command: /osutop [user] [mode] [sort] [reverse] [page] [mapper] [mods] [detailed] [compact]
                        Example: sbr-top mrekk
                        Example: /osutop mode:osu sort:pp reverse:true page:2 mapper:sotarks mods:DTany detailed:true
Aliases: top

Options:
Name Type Required Description Choices/limits/formatting Default
User String/Integer/Mention False The username/id of the user. If omitted, the database will search for the user's name N/A Database
Mode String False The gamemode. If omitted, the database will search for the user's default gamemode. Defaults to osu! standard osu/taiko/fruits/mania Database/osu
Sort String False The sorting options for the plays. Defaults to pp. pp/score/recent/acc/combo/misses pp
Reverse Boolean False Whether or not to reverse the order of the plays True/False False
Page Integer False The page of the plays to retrieve 1-20 1
Mapper String False Filter to only show plays set on maps by this mapper N/A N/A
Mods String False Filter to only show plays set with these mods.
Add "any" to the end for mod combinations (DT = DT, DTany = DT, HDDT, HDDTHR, EZDT)		 N/A NM
Detailed Boolean False Whether or not to show extra details True/False False
Compact Boolean False Whether or not to use compact mode True/False False
