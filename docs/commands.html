General Commands

osu! commands

compare

Compares two scores.
For the message command, the user's most recent score will be compared against the next most recent score 

                        Command: sbr-compare
                  Slash Command: /compare [score id 1] [score id 2]
Option Type Required Description Choices
Score id 1 Integer True The id of the first score to compare N/A
Score id 2 Integer True The id of the second score to compare N/A

leaderboard

Retrieves the top plays on a map 

                        Command: sbr-leaderboard [id]
                  Slash Command: /leaderboard [id] [page] [mods]
Aliases: 
                        maplb, mapleaderboard
Option Type Required Description Choices
ID Integer False The id of the map. If omitted, the previous map will be used instead N/A
Page Integer False The page of the scores to show. Defaults to 1 1-20
Mods String False Filters the leaderboard to show these mods. N/A
map

Returns the information of a beatmap 

                        Command: sbr-map "[search]" [id] +[mods]
                  Slash Command: /map [id] [mods] [detailed]
Aliases: 
                        m
Option Type Required Description Choices
Search String False The search query for the map. ie artist, title, difficulty, etc. Overrides map ID N/A
ID Integer False The id of the map. If omitted, the previous map will be used instead N/A
Mods String False Mods to use for difficulty/pp calculations N/A
Detailed Boolean False Shows extra map details (ar in ms, od in ms, object radius etc.) True/False
osu

                    Command: sbr-osu [username] [detailed]
              Slash Command: /osu [username] [detailed]
Aliases: 
                    o, profile
Option Type Required Description Choices
User String/Integer/Mention False The username/id of the user. If omitted or a mention is used, the database will search for the user's name N/A
Details Boolean False Whether or not to show extra details True/False
osuset

Set's the user's name for osu! commands 

                        Command: sbr-osuset [user]
                  Slash Command: /osuset [user] [gamemode] [skin]
Option Type Required Description Choices
User String True The username of the user N/A
Gamemode String False The main gamemode of the user. Defaults to osu! standard osu/taiko/fruits/mania
Skin String False The main skin of the user N/A
osutop

Retrieves the top plays of the user 

                        Command: sbr-osutop [user]
                  Slash Command: /osutop [user] [mode] [sort] [reverse] [page] [mapper] [mods] [detailed] [compact]
Aliases: 
                        top
Option Type Required Description Choices
User String/Integer/Mention False The username/id of the user. If omitted, the database will search for the user's name N/A
Mode String False The gamemode. If omitted, the database will search for the user's default gamemode. Defaults to osu! standard osu/taiko/fruits/mania
Sort String False The sorting options for the plays. Defaults to pp. pp/score/recent/acc/combo/misses
Reverse Boolean False Whether or not to reverse the order of the plays True/False
Page Integer False The page of the plays to retrieve 1-20
Mapper String False Filter to only show plays set on maps by this mapper N/A
Mods String False Filter to only show plays set with these mods.
Add "any" to the end for mod combinations (DT = DT, DTany = DT, HDDT, HDDTHR, EZDT)		 N/A
Detailed Boolean False Whether or not to show extra details True/False
Compact Boolean False Whether or not to use compact mode True/False

rs

Shows the most recent play(s) of the user 

                        Command: sbr-rs [user]
                  Slash Command: /rs [user] [page] [mode] [list]
Aliases: 
                        recent, r
Option Type Required Description Choices
User String/Integer/Mention False The username of the user. If omitted, the database will search for the user's name N/A
Page Integer False Which score to retrieve (lower number = more recent) 1-100
Mode String False The gamemode. If omitted, the database will search the user's default gamemode. Defaults to osu! standard osu/taiko/fruits/mania
List Boolean False Enables list mode. Only accuracy and ranking is shown True/False

Accuracy Float False The accuracy of the play. Defaults to 100.00 0-100
Combo Integer False The maximum combo of the play. Defaults to NaN 0+
