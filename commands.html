Commands

formatted as "commandName <required> [optional]".
options shown as `-name` should be used as follows: `-name arg` or just `-name` if boolean.
ie. `-page` can be `-page 3`.
Since `-reverse` is boolean, its just used as `-reverse`.


options shown as `(-name)` should be used as follows: `-arg`.
ie. `(-mode)` can be either `-osu`, `-taiko`, `-ctb` or `-mania`.
